# Portfolio Sandro Romaddana

Portfolio pribadi Sandro Romaddana untuk menampilkan pengalaman, proyek, dan bukti kerja di bidang Public Relations, content strategy, event communication, dan komunikasi pemasaran digital.

## Fokus Portfolio

- Public Relations
- Content Strategy
- Event Communication
- Digital Marketing Communication
- Copywriting dan pengelolaan konten media sosial

## Proyek Utama

- Tirta Rhea In House Magazine untuk BUMDes Amanah Karangsuko
- Crisis Management bersama Sosialoka
- Festival Kreasi Sampah bersama Bank Sampah Malang
- Skripsi tentang implementasi komunikasi pemasaran digital pada media sosial startup pengelolaan sampah

## Teknologi yang Digunakan

Portfolio ini dibangun menggunakan template React DeveloperFolio yang sudah disesuaikan untuk kebutuhan portfolio komunikasi dan public relations.

Teknologi utama:

- React
- Sass
- GitHub Pages
- JavaScript

## Cara Menjalankan Project

```bash
npm install
npm start
```

Untuk membuat build produksi:

```bash
npm run build
```

Untuk deploy ke GitHub Pages:

```bash
npm run deploy
```

## Catatan

Repository ini bukan portfolio software developer. Fokus utamanya adalah bukti kerja komunikasi, strategi konten, event, dan public relations.
